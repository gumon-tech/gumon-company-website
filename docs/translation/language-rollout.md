# Language Rollout Plan

## Phase 0 (live now)

- `th` (Thai)
- `en` (English)

## Phase 1 (target global launch)

- Asia: `ja`, `ko`, `zh-CN`, `zh-TW`, `vi`, `id`, `ms`, `hi`
- Europe: `de`, `fr`, `es`, `it`, `nl`, `pl`, `sv`, `uk`, `ru`, `tr`
- Americas: `es`, `pt-BR`, `en`
- Other regions: `ar`, `he`

## Rollout rule

- Add a locale route only when these are ready:
  1. Navigation labels translated
  2. Page metadata translated
  3. Legal and company information reviewed
  4. QA checklist passed on desktop + mobile

## Fallback behavior

- Unsupported content displays English with locale-specific fallback notice.
