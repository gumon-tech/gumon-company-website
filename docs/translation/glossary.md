# Gumon Glossary (v1)

Use these canonical terms consistently across all languages.

| Domain | Canonical EN | Canonical TH | Rule |
|---|---|---|---|
| Brand | Gumon Technology Co., Ltd. | บริษัท กุมอน เทคโนโลยี จำกัด | Never abbreviate legal name in legal pages |
| Product | Gumon Platform | แพลตฟอร์ม Gumon | Keep "Gumon" in Latin script |
| Delivery | Delivery Cadence | รอบการส่งมอบ | Use operational meaning, not literal shipping |
| Architecture | Application Layer | ชั้นงานแอปพลิเคชัน | Keep layer naming consistent across docs |
| Architecture | API Service Layer | ชั้นบริการ API | Keep API uppercase |
| Architecture | Data Stream Layer | ชั้นข้อมูลแบบสตรีม | Avoid translating as batch pipeline |
| Quality | Shared Standards | มาตรฐานร่วม | Always indicate cross-team consistency |
| Governance | Quality Gates | เกณฑ์คุณภาพ | Use measurable checkpoint meaning |
| Ecosystem | Partner Network | เครือข่ายพาร์ตเนอร์ | Refers to delivery ecosystem |
| Support | Knowledge Base | คลังความรู้ | Prefer this over "Wiki" in UI labels |

## Reserved tokens

- Keep unchanged: `Gumon`, `API`, `CLI`, `GitHub`, `DKS Center`, `Gumon Work`
- Email addresses and URLs must not be translated.
